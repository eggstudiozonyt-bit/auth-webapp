# Auth Web App

## Setup and Run Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/eggstudiozonyt-bit/auth-webapp.git
   cd auth-webapp
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Create a .env file**:
   - Copy the template `.env.example` to `.env` and update the values as needed.

4. **Run the application**:
   ```bash
   npm start
   ```

5. **Access the application**:
   - Open your browser and go to `http://localhost:3000`.

## Additional Information
- Ensure you have Node.js and npm installed on your machine.
- For more advanced setup options, refer to the documentation in the repository.
